# Precision Stockpile Control

Precision Stockpile Control is a RimWorld 1.6 mod planned around opt-in, low-overhead stockpile controls.

## Planned Features

- Per-item stockpile target maximums and refill thresholds
- Batch hauling rules and feeder source/destination links
- Fine-grained stockpile ordering within vanilla priority bands

## Development Status

This repository currently contains the initial scaffold only. Implementation code has not been added yet.

The authoritative design is in `04_PSC_DESIGN.md`.
